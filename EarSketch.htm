setEffect
( track , effectType , effectParameter , effectValue )
This function applies an effect to a specified track number and sets a parameter of that effect to a particular value for the entire track. For detailed information on all of the effects available to use with setEffect(), please see Every Effect Explained in Detail in the curriculum.
Parameters:
track :  Integer
Track to place effect onto (or MASTER_TRACK to apply it to all tracks)
effectType :  Effect Constant
Effect constant: BANDPASS, CHORUS, COMPRESSOR, DELAY, DISTORTION, EQ3BAND, FILTER, FLANGER, PAN, PHASER, PITCHSHIFT, REVERB, RINGMOD, TREMOLO, VOLUME, or WAH
effectParameter :  Integer
Constant indicating which parameter of the effectType to create the envelope for. (See Every Effect Explained in Detail in the curriculum sidebar for a complete list of effect parameters.)
effectValue :  Float
Value of effect parameter
Example:
# Apply a delay effect on track 1
setEffect(1, DELAY, DELAY_TIME, 250)
setEffect
( track , effectType , effectParameter , effectStartValue , effectStartLocation , effectEndValue , effectEndLocation )
This function applies an effect to a specified track number. Unlike the other version of setEffect(), there are additional arguments for setting an envelope that changes the value of an effect parameter over time. For detailed information on all of the effects available to use with setEffect(), please see the curriculum: Every Effect Explained in Detail
Parameters:
track :  Integer
Track to place effect onto (or MASTER_TRACK to apply it to all tracks)
effectType :  Effect Constant
Effect constant: BANDPASS, CHORUS, COMPRESSOR, DELAY, DISTORTION, EQ3BAND, FILTER, FLANGER, PAN, PHASER, PITCHSHIFT, REVERB, RINGMOD, TREMOLO, VOLUME, or WAH
effectParameter :  Integer
Constant indicating which parameter of the effectType to create the envelope for. (See Every Effect Explained in Detail in the curriculum sidebar for a complete list of effect parameters.)
effectStartValue :  Float
Value of effect parameter at beginning of effect envelope
effectStartLocation :  Float
Location of the effectStartValue
effectEndValue :  Float
Value of effect parameter at end of effect envelope
effectEndLocation :  Float
Location of the effectEndValue
Example:
# Change filter cutoff frequency from 100Hz to 2000Hz over measures 1 to 3

Making Custom Beats: makeBeat

makeBeat() allows us to compose music note by note instead of at the measure level, perfect for drum beats. This approach is often called step sequencing in music production.

Strings

We need to understand the string data type to use makeBeat(). In Python, a string is a series of characters with single or double quotation marks around it, like "Hello World!" or 'This is a test sentence'. Strings are often used in programming to represent non-numerical data such as words, but can include numbers. For example, the string "123 Ferst Dr." could represent an address, using numbers, spaces, letters, and punctuation. Like numbers (and other types of data), strings can be assigned to variables: address = "123 Ferst Dr."

Beat Patterns with Strings

We use strings with makeBeat() to define rhythmic patterns, which we call beat patterns. Beat pattern strings in EarSketch use characters to refer to sixteenth note sub-beats of a measure. A character is a single unit of information, and can be a letter, number, whitespace, punctuation, or symbol. The configuration of characters allow us to place clips at specific places in the measure and define the clip’s play length. Here is an example of a beat pattern string, assigned to a variable called myDrumBeat:

myDrumBeat = "0-00-00-0++0+0"

  • 0 starts playing a clip.

  • - is a rest, meaning silence.

  • + extends the audio clip into the next sixteenth note sub-beat. It always follows a 0 or +.

A 16-character beat string comprises a measure, but beat strings do not have to be this length; the beat string "0+++", lasting for one beat, is perfectly acceptable. EarSketch reads the string from left to right, adding the chosen clip to the DAW according to the string’s rhythm. The myDrumBeat string, above, creates the following rhythm in the DAW:

makeBeat()

makeBeat() takes four arguments:

  1. Clip Name

  2. Track Number

  3. Measure Number: makeBeat() only requires a starting measure; the string length determines the end measure.

  4. Beat String

Creating an interesting beat is important, but you will also want to have control over how many times the beat repeats throughout your song. You could specify each and every measure you want the rhythm to play, or you can let the code do some of the heavy lifting for you with for-loops.

# python code
#
# script_name: Multi Beat
#
# author: The EarSketch Team
#
# description: Using several makeBeat calls, and overlapping rhythms
#
#
#

#Setup

from earsketch import *
init()
setTempo(120)

#Music

synth = DUBSTEP_FILTERCHORD_002
cymbal = OS_CLOSEDHAT01
beat1 = "-00-00+++00--0-0"
beat2 = "0--0--000--00-0-"

for measure in range(1,4):
   makeBeat(synth, 1, measure, beat1)
   makeBeat(cymbal, 2, measure, beat2)

#Finish

finish()

// script_name: Multi Beat

// author: The EarSketch Team

// description: Using several makeBeat calls, and overlapping rhythms

//Setup

init();
setTempo(120);

//Music

var synth = DUBSTEP_FILTERCHORD_002;
var cymbal = OS_CLOSEDHAT01;
var beat1 = "-00-00+++00--0-0";
var beat2 = "0--0--000--00-0-";

for (var measure=1; measure < 4; measure++) {
   makeBeat(synth, 1, measure, beat1);

}

Chapter 12 Summary

  • A string is a data type that consists of a series of characters encapsulated by single or double quotes. Strings are most often used to represent non-numerical data.

  • Strings are used with the makeBeat() function to create rhythmic patterns in EarSketch. makeBeat() takes a beat string to define each sixteenth note sub-beat of its pattern. A 0 starts playing a clip, a + extends the note for the next sub-beat, and - creates a rest.

  • makeBeat() takes four arguments:

    • clipName: The clip a beat is constructed from.

    • trackNumber: The track on which music is placed.

    • measureNumber: The starting measure of the beat. The beat string determines the total length.

    • beatString: A string that specifies the rhythm created.
